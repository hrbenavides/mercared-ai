# MercaRed AI

**Ecosistema digital de abastecimiento B2B** para pequeños comercios, distribuidores y transportistas.

## Características
- Roles: Comprador, Vendedor, Transportista, Administrador.
- Catálogo de productos con carrito de compras.
- Gestión de pedidos con seguimiento de estado.
- Notificaciones en tiempo real.
- Dashboard por cada rol.

## Tecnologías
- Flask (Python)
- SQLite
- HTML + CSS

## Cómo ejecutar localmente
1. Clona el repositorio.
2. Ejecuta `python start.py` (instalará dependencias automáticamente).
3. Abre http://127.0.0.1:5000
